サイトを“人格化”して状態を可視化。直感でわかるサイト診断プラグイン。

== Description ==

あなたのサイト、今どんな状態？??

Site Personality Analyzer は、サイトの状態を分析し、
「人格」と「Google視点」で分かりやすく診断するプラグインです。

数値や専門用語に頼らず、
直感的にサイトのコンディションを把握できます。

= 主な機能 =

* サイト人格診断（5段階）
* Google視点評価（S?D）
* 改善ポイント自動抽出
* ランダムコメント表示

= サイト人格 =

スコアに応じて以下のように判定されます。

* 最強
* 優秀
* 普通
* 不調
* 危険

= Google視点評価 =

スコアをもとに以下の評価を表示します。

* S / A / B / C / D

= 改善ポイント =

以下のような問題を自動検出します。

* 記事数が少ない
* 更新が止まっている
* 内部リンクが不足している

= このプラグインの特徴 =

このプラグインは「分析ツール」ではなく、
**“理解して行動できること”**を目的としています。

= 想定ユーザー =

* サイト運用者
* ブロガー
* Web担当者
* クライアントワークをしている人

== Installation ==

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面から有効化
3. 「サイト診断」メニューから利用

== Frequently Asked Questions ==

= スコアはどのように計算されますか？ =

以下の要素をもとに算出しています。

* 投稿数
* 更新頻度
* 内部リンク数

= 数値の正確性はどれくらいですか？ =

厳密な分析ではなく、サイト状態を把握するための目安です。

= フロントには影響ありますか？ =

ありません。管理画面のみで動作します。
